Projects

Bejeweled Blitz

Bejeweled Blitz is a classic tile-matching puzzle game developed in C++. The game challenges players to swap adjacent gems to form horizontal or vertical lines of three or more gems of the same color, which then disappear, allowing more gems to fall into place. The primary objective is to score as many points as possible within a limited time frame.

Minesweeper

Minesweeper is a classic logic-based puzzle game developed in C++. The goal of the game is to clear a rectangular board containing hidden mines without detonating any of them, with the help of clues about the number of neighboring mines in each field.

Operating System Shell

The Custom Operating System Shell is a command-line interface (CLI) developed in C that allows users to interact with the underlying operating system by executing commands. This shell provides fundamental functionalities similar to those found in standard shells like Bash or Zsh, allowing for command execution, file manipulation, and process control.
